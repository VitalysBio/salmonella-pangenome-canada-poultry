## Panaroo pangenome analysis

Pangenome analysis was performed using Panaroo v1.5.2 with Biopython v1.78. Panaroo was applied to the genome annotations generated by Prokka in order to identify core and accessory gene clusters across all genomes included in the study.

### Software environment

Panaroo was executed in a environment with the following versions:
- Panaroo v1.5.2
- Biopython v1.78

### Input data

Panaroo was run using GFF files generated by Prokka. These files were collected from the Prokka output directories and copied into a new folder prior to pangenome analysis.
